# Project Context
Build a database schema for a mechanical keyboard price tracker.

# Tech Stack
SQLite

# Core Requirements
- Create a table for Keyboard_Items (id, name, brand, retail_price).
- Create a table for Price_History (id, item_id, aftermarket_price, date_recorded, source_url).
- Ensure there are proper relational links between the items and their pricing history.
